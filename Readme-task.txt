1.Firstly make a connection to the server
2.Make sure your terminal is in the server directory. To be in server directory run "cd server" in terminal.
3.After that establish a connection using "nodemon server"
4.Run "npm start" in the new terminal to run the app.
5.After that, you will be redirected to the home page of the website.
6.So for seeing the tasks click on "Task" in the navigation bar.
7.To create tasks go to the home page and then click on the "Create Task" button.
8.Fill the form for creating the task.
9.You can filter tasks in the "Tasks" section using search and filter menus.
10.You can use different utilities for each task such as complete status, delete and edit.
